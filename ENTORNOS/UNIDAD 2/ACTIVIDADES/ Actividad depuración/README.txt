Cada uno de vosotros tiene que depurar y corregir 2 programas con varios errores de un proyecto con varios programas que yo os voy a pasar. El primero tiene que estar entre los programas [1,6] y el segundo entre los [7,12]. Procurad no repetir con los compañeros el mismo programa.

La entrega consistirá en realizar un video grabando (uno solo) la pantalla mientras describís , para cada programa, los errores existentes usando el depurador y la forma de corregirlos. La mecánica debe ser la siguiente:

Ejecutar el programa. El programa no funciona porque el resultado debería ser X y sale Y.
Colocar alguno/s punto/s de ruptura/s donde consideréis oportuno.
Avanzáis con el depurador y cuando una variable tiene un valor que no cuadra comentáis el por qué.
Volvéis a ejecutar el programa y el resultado es correcto, os despedís y el video termina, si el resultado sigue siendo incorrecto volvéis al punto 1.
Alternativamente, podéis entregar una memoria con una descripción exhaustiva de este proceso.
